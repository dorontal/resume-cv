RECENT PROJECTS
  • Automatic robot design: use optimization to evaluate different physical robot designs according to how well the robots embodying these designs perform in simulation. xThis helps tailor the robot to its environment before it is built.
  • Automatically learning robotic controllers: use parameter optimization methods to automatically select symmetry assignments (servo couplings) in the robot controller. This greatly speeds up machine learning on the robot.
  • Three dimensional vision-based SLAM: A solution to the robotic mapping problem in three dimensions, using a single grayscale camera, a single light source, and Bayesian inference.
  • Robotic locomotion: Work done in collaboration with Andrew Ng, Gregory Williams: we used simple reinforcement learning direct search methods to train a four-legged robot in various locomotion gaits. Here are some results with the first version of the robot, at UC Berkeley's Soda Hall.

HARDWARE ACCOMPLISHMENTS
  • Yellowbot: This robot was built as part of collaboration with Andrew Ng and Gregory Williams at UC Berkeley to test machine learning algorithms for quadruped walking. In 2003, it was donated to Stanford’s robotics lab.
  • Rough Terrain Quadruped (RTQ): First outdoor version, developed at NASA Ames Research Center. This robot was used to study the usage of symmetry in walking control. It is also physically symmetric along several axes: front/hind, left/right, up/down. It has 3 accelerometers, 3 gyroscopes, 4 sonars, 4 light sensors, 1 temperature sensor for automatic temperature-calibration, 4 force-sensing resistors on the feet, 12 current sensors for each servo motor, bluetooth wireless and 80 MIPS of computing via 5 networked AVR microcontrollers.
  • Rough Terrain Quadruped II (RTQ2). RTQ improved. The main changes include stronger lower legs, stiffer spring-suspension lower-leg shock mechanisms, on-board 400 MHz Linux computer with WiFi wireless communications, higher-resolution and faster servo controller, Lithium-Polymer batteries and a sealed box for power and electronics.
